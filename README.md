# My little dev blog

## Project setup

1. Install Node v14.17.5
2. Cry because you just installed node.
3. Install npm
4. From the root run 

```
npm install
```

### Run Locally
```
npm run serve
```

### Lints and fixes files (run this often)
```
npm run lint
```

### Deploy to Github Pages

Manually:
```
npm run deploy
```

Automatically:
Just push to master and let github workflows deal with it
